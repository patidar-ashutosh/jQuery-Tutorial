What is jQuery? jQuery javascript ka lightweight library hai jQuery ham jo code karte hai wo sab javascript ka code he hota hai but jQuery me vo code thoda sa small ho jata hai ab ham javascript ka use kare ke code karte hai to kya problem aati thi ? 1) Long Selector => hame DOM ko target karne ke liye long selector likhne padte the. 2) Complex Animations => Complex Animations create karne padthi thi jise code Lengthy ho jata thaa. 3) Lengthy DOM Manipulation => jab ham DOM ka use karke value SET ya GET karte the to code Lengthy ho jata thaa. 4) Lengthy Ajax Coding => javascriptse Ajax ka code Lengthy ho jata thaa ye sab problem ko slove karne ke liye jQuery ki team ne is sabhi problem ka ek-ek small function create kar diye or us sabhi function ko collect karke ek file me store kar diya jise ham jQuery kehte hai. Why use jQuery? jQuery ko use karne ke benefits :- 1) Short Selector => jo selector hai wo short ho jate hai 2) Variety of Animation function => is me Animations karne ke liye Variety of function hai 3) Easy DOM Manipulation => DOM ka use kare HTML me koi value SET ya GET karna hai to easy ho jata hai 4) Easy CSS Styling => CSS ka Styling karna easy ho jata hai 5) Easy DOM Traversing => kisi bhi element ko target karna easy ho jata hai 6) Simple Ajax Code => Ajax ka code ham aasani se kar sakte hai (1 line me hi 10 line ka code kar sakte hai) example jese ki ham class name se target karne ke javascript me likhte the document.getElementsById("classname"); lekin use jQuery me ham Simple likhe ge $(".classname ") benefits of jQuery Browser Independent => jQuery sabhi browser ko support karta hai Increase Coding Speed => code karne ki speed javascript ke compare me high ho jata hai
